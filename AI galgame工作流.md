# AI Galgame 工作流

> 按编号顺序执行，每一阶段产出作为下一阶段的输入。

---

## 第一阶段：企划与设定
1. **01_项目企划** → 确定主题、世界观、目标受众
2. **02_主要角色设定** → 主角性格、外貌、背景故事
3. **03_配角设定** → 配角基本信息与关系网

## 第二阶段：故事架构
4. **04_剧情大纲** → 整体故事起承转合
5. **05_共通线章节细纲** → 共通线各章节情节拆解
6. **06_个人线章节细纲** → 各角色个人线章节拆解
7. **07_分支选项** → 关键分支点与好感度逻辑

## 第三阶段：资产规划
8. **08_角色立绘** → 立绘需求文档（姿势/服装）
9. **09_共通线场景细纲** → 共通线场景列表与演出需求
10. **10_个人线场景细纲** → 个人线场景列表与演出需求
11. **11_表情差分清单** → 各角色表情差分需求汇总
12. **12_音频清单** → BGM + SE 需求列表

## 第四阶段：资产制作
13. **13_表情差分** → 表情差分成品文件
14. **17_音频资源** → BGM / SE 音频文件
15. **18_配音资源** → 角色配音文件

## 第五阶段：剧本撰写
16. **14_共通线场景剧本** → 共通线完整剧本（含演出标记）
17. **15_个人线场景剧本** → 各角色个人线剧本（含演出标记）

## 第六阶段：工程集成
18. **16_game_project** → 导入引擎，整合所有资产与剧本，打包发布
