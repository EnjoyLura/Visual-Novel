# 剧情大纲

> **时间跨度**：3月底 ~ 7月底（约四个月）
> **结构**：12章线性推进，社团招新（第3章）与打工（第6章）为两个关键分支点
> **情感弧线**：初遇 → 深入 → 文化祭高潮 → 花火大会升温 → 退缩 → 告白 → 结局

## 二、各章剧情大纲

### 第一章：开学前后（3月底·2日）

| 项目 | 内容 |
|------|------|
| **时间** | 开学前一天 + 开学第一天 |
| **路线** | 全路线共通 |

**第1日 · 开学前一天**
- 下午，悠真被母亲指派出门买酱油。在便利店偶遇打工的篠崎茜。
- 买完酱油绕路闲逛，在樱花树下重逢七瀬千纱——阔别五年的青梅竹马，就住在隔壁。对话生疏，有一定距离感。
- 夜晚，悠真在房间内的独白。

**第2日 · 开学第一天**
- **校门口**：被星野千夏撞上——铁三角（悠真·千夏·信）新学期重聚。三人同班。
- **始业式**：白鸟绫音以学生会长身份致欢迎词。
- **教室**：千纱作为转学生被介绍进班。教室后三人（千夏、信、悠真）闲聊叙旧。
- **学生会搬书**：悠真被指派去学生会帮忙搬书，与绫音有简短接触。
- **放学后**：悠真与小春在房间闲谈。

| 项目 | 内容 |
|------|------|

| **路线** | 全路线共通 |

**第二天**
- **上学·河堤**：上学路上河堤偶遇千纱，同行到校。
- **课间·教室后方**：铁三角（悠真·千夏·信）的固定闲聊圈。

- **放学后·归途**：信去音乐部、千夏去田径部，两人告别后留下悠真一人回家。到家发现小春也在客厅，小春问起「哥哥没有社团吗？这么早回来。」

**第三天**
- **上学·河堤**：再次偶遇千纱同行。
- **课间·教室**：信「叛徒审问」——他看到了悠真和千纱一起上学，开始八卦追问。

- **放学后·操场**：偶遇千夏训练。

**周五**
- **夜晚·房间**：小春和悠真房间夜话。

**周末（第一个选项分支）**
| 选择 | 场景 |
|------|------|
| 出门 | 小春让悠真顺便去便利店帮买巧克力和薯片。出门可能触发额外偶遇 |
| 留在家 | 小春进悠真房间看漫画，看哥哥打游戏 |

**下周一**
- **教室**：千纱当选图书室委员。班主任预告社团招新即将开始。
- **放学后·河堤**：偶遇千纱同行回家。
- **夜晚·房间**：悠真做出微小决定——了解一下社团，是时候告别「回家部」了。

## 三、主题贯穿

> **「从为别人而存在，到为自己做出选择。」**

- 第1~9章：悠真扮演「支持者」——帮助、陪伴、观察。感情在积累但未确认。
- 第10章（退缩）：感情已清晰到无法忽视，但本能地退缩——「维持现状更安全」的诱惑。
- 第11~12章：跨过退缩，选择彼此。五条线的共同终点：悠真不再只是「帮她」，而是「选择了她」。
- **单身线**：反主题对照——悠真从未为自己做出选择，故事在花火大会的烟花声与睡意中悄然结束。
