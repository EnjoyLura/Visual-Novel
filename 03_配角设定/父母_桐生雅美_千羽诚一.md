# 桐生雅美 & 千羽诚一（男主父母）

> **定位**：重组家庭的双亲 · 温暖而开明的家庭基石
> **姓氏说明**：男主随母姓「桐生」｜小春暂用「千羽」，父亲名为「千羽诚一」，姓氏一致性待后期统一 ✅

---

## 桐生雅美（Kiryuu Masami）—— 母亲

### 基本档案
| 项目 | 内容 |
|------|------|
| **身份** | 男主的生母、小春的继母 |
| **年龄** | 40岁前后 |
| **职业** | 全职上班族（公司职员） |
| **性格关键词** | 开明、开朗、朋友型、爱开玩笑 |

### 外貌速写
- 和男主有相似的五官轮廓——温和的眉眼，笑起来眼睛会弯。短发或及肩中长发，打理得干净利落（上班族风格）。
- 在家穿休闲服时会显得比上班时年轻不少。身材保持得很好，看起来比实际年龄小几岁——同学来家里玩时常被说「你妈妈好年轻」。

### 性格速写
- **和儿子像朋友**：从不摆家长的架子。会和男主开玩笑、会八卦他的感情状况、会在饭桌上说「有喜欢的女生要带回来给妈看看」——然后满意地看着男主耳根发红。
- **开明但不放任**：尊重孩子们的选择，但也不是什么都不管。如果发现男主真的在烦恼，她会收起玩笑，认真坐下来和他聊——「妈妈不是要替你决定，只是想让你知道，你说什么我都听。」
- **家庭的气氛核心**：是这个重组家庭中最重要的「胶水」。继父沉默寡言，她用开朗和温暖把四个人黏在一起。晚饭餐桌上的笑声，十次有八次是她带起来的。
- **对小春的态度**：把小春当作亲生女儿来疼。从不会区分「这是我的孩子」「那是你的孩子」——对她来说，两个都是她的孩子。
- **如果知道小春和男主的感情**：可能不会太震惊——她太了解自己的孩子们了。第一反应大概是：「……我就说嘛。」然后认真地告诉两人——「你们幸福比什么都重要。」

### 剧情功能
- 家庭日常场景的核心人物——晚饭、周末、节日等场合的「温度来源」。
- 在小春线中起到「缓冲」作用——当兄妹变恋人的事情浮出水面时，她的态度决定家庭反应的下限。
- 偶尔在共通线中「不经意的助攻」——「儿子，今天便利店那个女孩是不是你们学校的？」（男主：你怎么知道？！）

## 千羽诚一（Chiba Seiichi）—— 父亲

### 基本档案
| 项目 | 内容 |
|------|------|
| **身份** | 小春的生父、男主的继父 |
| **年龄** | 40岁前后 |
| **职业** | 普通上班族，朝九晚五（偶尔加班） |
| **性格关键词** | 寡言、务实、可靠、观察者 |
| **特殊关系** | 与七瀬千纱的父亲是老朋友 |

### 外貌速写
- 中等身材，不胖不瘦。脸上有些岁月的痕迹——眼角细纹、鬓角几根白发。表情偏少，不笑的时候看起来有点严肃，但其实只是不习惯把情绪写在脸上。
- 上班时穿衬衫西裤，回家换上一件洗得发旧的運動衫。戴着最普通的电子表——是小春小学时送的父亲节礼物。

### 性格速写
- **沉默的行动派**：话很少，但做的事很多。男主房间的灯坏了——隔天就修好了。小春提了一句「最近想吃布丁」——冰箱里就出现了布丁。他从来不邀功，甚至不一定会说「是我买的」。
- **笨拙的关心**：想表达关心时词不达意。「今天在学校怎么样」说出来变成了「没给老师添麻烦吧」。被母亲吐槽后微微脸红——「……我知道自己的意思就行了。」
- **务实的家庭守护者**：不干涉孩子们的选择，也从不评判。他的信条是——「不管发生什么，这个家不能散。」不是用嘴说的，是用每一天的行动做的。
- **对男主的鼓励**：偶尔会简单地说一句「加油」——措辞简单到几乎敷衍，但配合他少有的认真眼神和拍在肩膀上的手，分量完全不一样。男主知道，继父的「加油」等于别人的一千句话。
- **对小春的感情**：女儿是他的软肋。虽然嘴上不常说，但小春是他的全世界。如果知道小春和男主的感情——他会沉默比平时更久，然后说：「你们想好了吗？」确认两人是认真的之后，他不会反对。他最大的担忧从来不是「兄妹在一起不合适」，而是「如果这份感情受伤了，小春会受伤」。
- **与七瀬家的关系**：和千纱的父亲是老朋友——这在千纱线中是一个重要的连接点。当年两家一起带孩子出去玩的日子，他至今还记得。千纱一家搬回来后，他可能是最早知道消息的人之一。

### 剧情功能
- 家庭场景中「静」的那一面——和母亲的「动」形成平衡。
- 在男主迷茫时提供「沉默的支持」——不需要长篇大论，一句「加油」就够了。
- 在千纱线中作为两家之间的纽带——「你千纱叔叔说……」
- 在小春线中是「父亲」这一身份的情感锚点——他不需要说太多，但他的态度至关重要。

## 父母互动模式

| 场景 | 典型互动 |
|------|----------|
| **晚饭** | 母亲说个不停（今天公司发生了什么、邻居家怎么了、你们学校呢），父亲默默吃饭偶尔「嗯」一声，男主和小春交换一个「又来了」的眼神。 |
| **周末** | 母亲拉着全家去超市采购——「一家人一起走！」父亲跟在后面推购物车，男主和小春走在中间。看起来像最普通的家庭——因为本来就是。 |
| **发现孩子们有心事** | 母亲会直接问「怎么了怎么了」，父亲保持沉默但竖起耳朵。等母亲问完了，父亲在走廊里碰到孩子，只说一句：「……有什么要帮忙的，就说。」 |

## 写作参考

### 母亲
- **语言风格**：口语化、语气词丰富、句子较短。说话时经常带笑声（「あはは」「もう〜」）。对男主不带敬语，对小春温柔但不刻意。
- **标志性台词**：
  - 「いいじゃん、若いんだから！」（有什么不好，年轻嘛！）
  - 「で、気になる子とかいるの？」（所以，有没有在意的女生啊？）
  - 「お母さんはね、あなたたちが幸せならそれでいいの。」（妈妈啊，只要你们幸福就好了。）

### 父亲
- **语言风格**：句子极短，有时只有一个词。话少到需要旁白补充他的行动和表情。偶尔说出的长句往往是经过深思熟虑的——且很有分量。
- **标志性台词**：
  - 「……頑張れ。」（……加油。）
  - 「大丈夫か。」（没事吧。）
  - 「お前たちが決めたことなら、俺は何も言わない。」（如果是你们自己决定的，我没什么好说的。）
